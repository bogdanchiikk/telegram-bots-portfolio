# 🤖 Telegram-боты (Python)

Портфолио проектов: боты на Python для Telegram.  
Проекты разработаны во время стажировки в MarkOnlineStudio. Код в закрытых репозиториях — ниже описание и используемый стек.

---

## Проекты

### 1. Бот-переводчик

Перевод текста и голосовых сообщений на разные языки и обратно. Пользователь отправляет текст или голосовое — бот распознаёт речь (при необходимости), переводит на выбранный язык и при необходимости переводит ответ обратно.

**Стек:** Python · python-telegram-bot · API распознавания речи · API перевода

---

### 2. Нанабананабот

Переводчик на несколько языков с возможностью перевода обратно на исходный язык. Удобный выбор языка, обработка текста и голосовых сообщений, логика диалога и обработка ошибок.

**Стек:** Python · Telegram Bot API · сервисы перевода

---

### 3. OpenClaw — боты для групп

Набор ботов для работы в групповых чатах Telegram:

- **Основной бот** — в личке и в группе: ответы на текст и голосовые (голос → распознавание речи → текст → ответ по контексту). Чтение ссылок из сообщений.
- **Групповой бот** — обрабатывает все сообщения в группе (текст и голосовые), распознаёт голос через Whisper, формирует ответ.

Настройка прав и приватности для групп, деплой через systemd на сервер.

**Стек:** Python · python-telegram-bot · OpenAI Whisper · ChatGPT API · systemd · Linux

---

### 4. Бот-магазин мерча (футболки, толстовки)

Мини-интернет-магазин в Telegram: каталог товаров (футболки, толстовки, чехлы), генерация и редактирование дизайнов по текстовому описанию (AI), примерка дизайна на товаре (макапы), корзина и оформление заказа с доставкой и оплатой.

**Реализовано:**
- Генерация изображений по запросу (Gemini API), редактирование загруженных изображений
- Каталог: категории, размеры, цвета, цены; выбор типа (classic/oversize)
- Генерация макапов (примерок) — наложение дизайна на фото товара
- Разделы «Мои дизайны» и «Мои товары», добавление в корзину
- Оформление заказа: ФИО, регион, адрес, индекс, телефон, email; валидация
- Оплата через Robokassa, экспорт заказов в Google Sheets, промокоды, расчёт доставки
- Админ-бот: загрузка каталога, просмотр заказов

**Стек:** Python · python-telegram-bot · Gemini API · SQLAlchemy · Alembic · Robokassa · Google Sheets API · деплой (Beget)
